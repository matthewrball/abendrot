# Name clearance summary (why Abendrot)

Real-world brand-crowding read (not legal clearance; USPTO/EUIPO register UIs blocked automated fetch — for a free OSS personal project this is proportionate, but do a paid TESS/EUIPO pull before any registered/monetized use).

## Abendrot — **GO** (chosen)
- **No app/product/repo** named "Abendrot" exists in the screen/blue-light/circadian space (or anywhere notable) — the cleanest, near-empty namespace of every candidate.
- Best thematic fit by far: *Abendrot* = the red glow of sunset = literally what a screen-warming app does.
- **Rankable/ownable** — you could realistically be #1 for "Abendrot"; the genericness/SEO problem that sank Ruhe doesn't exist.
- Only cost: longer + first-glance pronunciation ("AH-bent-roht"), the price of distinctiveness.
- `abendrot.app` **purchased** (primary domain).

## Ruhe — GO-with-caution (passed over)
- No in-lane collision (the App Store "Ruhe" is kitchen/bath fittings, India), perfect meaning ("rest/calm"). BUT: everyday generic German word ("Ruhe!" = "Quiet!"), **`ruhe.app` already taken by an AI-healthcare company**, SEO-hostile, weak as a defensible mark, and a permanent ROO-uh pronunciation tax. Shippable but not ownable.

## Schimmer — GO-with-caution (passed over)
- Exact "Schimmer" spelling is clear, but the English **"Shimmer" homophone is crowded** (Shimmer Care ADHD app, FB's Shimmer library, etc.) and the Sch/Sh spelling split hurts search + recall.
